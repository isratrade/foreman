POST /api/bookmarks
Create a bookmark.

Examples

POST /api/bookmarks
{
  "bookmark": {
    "query": "bar",
    "public": false,
    "name": "foo-bar",
    "controller": "hosts"
  }
}
200
{
  "bookmark": {
    "query": "bar",
    "user": {
      "login": "admin",
      "name": "Admin User",
      "email": "admin@someware.com"
    },
    "public": false,
    "owner_type": "User",
    "name": "foo-bar",
    "owner_id": 135138680,
    "id": 980190963,
    "controller": "hosts"
  }
}

Params

Param name Description
bookmark
required
Value: Must be a Hash
bookmark[name]
required
Value: Must be String
bookmark[controller]
required
Value: Must be String
bookmark[query]
required
Value: Must be String

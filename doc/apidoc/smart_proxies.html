    "id": 980190962
  }
}

Params

Param name Description
id
required
Value: Must be an identifier, string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
>>>

POST /api/smart_proxies
Create a smart proxy.

Examples

POST /api/smart_proxies
{
  "smart_proxy": {
    "url": "http://server:8443",
    "name": "master02"
  }
}
201
{
  "smart_proxy": {
    "url": "http://server:8443",
    "created_at": "2012-11-13T13:49:26Z",
    "updated_at": "2012-11-13T13:49:26Z",
    "name": "master02",
    "id": 980190963
  }
}

Params

Param name Description
smart_proxy
required
Value: Must be a Hash
smart_proxy[name]
required
Value: Must be String
smart_proxy[url]
required
Value: Must be String

POST /api/lookup_keys
Create a lookup key.

Examples

POST /api/lookup_keys
{
  "lookup_key": {
    "key": "testkey",
    "is_param": true
  }
}
201
{
  "lookup_key": {
    "required": false,
    "override": false,
    "id": 980190963,
    "key": "testkey",
    "default_value": null,
    "description": null,
    "is_param": true
  }
}
